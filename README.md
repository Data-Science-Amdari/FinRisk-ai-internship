# FinRisk AI Internship Project

## 📌 Overview
This project is focused on financial risk analysis and fraud detection in UK banking datasets.  
It includes synthetic datasets, exploratory data analysis (EDA), feature engineering, and machine learning models for:
- **Credit Scoring** (XGBoost, Logistic Regression baseline)
- **Fraud Detection** (Isolation Forest, One-Class SVM)
- **Feature Engineering** (statistical transformations, binning, interactions)
- **Validation** (time-based splits, cross-validation, out-of-time testing)
